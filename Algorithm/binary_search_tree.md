# BST (Binary Search Tree)
- Data들을 빠르게 검색할 수 있도록 ${\textsf{\color{red}체계적으로 저장}}$을 해 두고, 최대 ${\textsf{\color{red}O(logN)의 빠른 속도로 값을 검색}}$할 수 있는 자료구조

## BST 성능
- 삽입 : 평균 O(logN)
- 삭제 : 평균 O(logN)
- 탐색 : 평균 O(logN)
- 트리의 높이만큼 시간이 걸림 O(h)
- 이진 트리가 균형적으로 생성되어 있는 경우 O(logN)
- 편향 이진 트리의 경우 O(N)
